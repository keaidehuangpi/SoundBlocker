# SoundBlocker
Simply blocks specific sound you don't want.  
Use /soundblocker in game to open the gui.  
Configurable in game or in config file.
## Features
- Block specific sounds
- Edit blocked sounds
- Hide disabled sounds
- Normal and strict mode
- --Normal mode just blocks all the sounds with specific name.
- --Strict mode also checks if the pitch matches.
- ---and,the sounds to block in different modes are not shared.
